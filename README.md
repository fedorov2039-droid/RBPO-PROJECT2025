Сервис Кинотеатра (Cinema Service)

Проект разработан в рамках дисциплины "РБПО".

Сервис предоставляет REST API для управления работой кинотеатра.

Сущности:

1. Movie — Фильм (название, описание, длительность). 
2. Hall — Зал (название, вместимость). 
3. Screening — Сеанс (связывает фильм и зал, имеет цену и время начала).
4. Customer — Покупатель.
5. Ticket — Билет (привязан к сеансу и покупателю).

Бизнес-операции (Logic):

1. Покупка билета: Проверяет вместимость зала. Если мест нет — покупка запрещена. 
2. Групповая покупка (Транзакция):** Покупка сразу нескольких билетов. Если хоть один не проходит — отменяется вся покупка.
3. Возврат билета: Разрешен только если до начала сеанса осталось больше 1 часа. 
4. Отмена сеанса: Удаляет сеанс и автоматически аннулирует все билеты (возврат средств).
5. Валидация данных: Проверка цен, времени и вместимости.

Технологии:
Java 21
Spring Boot 3(Web, Data JPA)
PostgreSQL (Реляционная база данных)
Maven
